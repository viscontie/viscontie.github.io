Measuring Narrative Change

Project Outcome

Dashboard and protocol that measures narrative change through analyzing and visualizing social media exports and state reported data, sentiment analysis of social media engagement, and thematic trend analysis using NLP tools.

Future Use

Future Built a protocol for updating the data every 6 months so that with each new batch of social media exports, STEM Next can upload new csv files to autogenerate new analyses and graphs and track narrative change over years.

Impact

Created a system for tracking narrative change at STEM Next and help them show that they are working towards their goal of reaching 1.5 millions kids

Deliverables

My project included...

  • Website displaying and contextualizing narrative change data for STEM Next employees, partners, and funders.
  • Protocol for updating new data files every 6 months to automatically update Tableau visualizations, run sentiment analysis scripts, and thematic coding scripts.
  • Hour long presentation on August 20th to STEM Next employees about my findings and the trajectory of measuring narrative change moving forward

Technical skills

  • NLP (Roberta) to automate thematic analysis
  • Sentiment Analysis using NLP
  • Python/pandas/matplotlib for data analysis and visualization
  • Tableau visualizations and dashboard
  • Website/Dashboard design

Soft Skills

  • Communicating with a nontechnical audience
  • Project management
  • Adaptibility: pivoting project due to social media privacy policies
  • Collaboration with stakeholders
  • Framing data to tell the story of narrative change

Thematic Analysis

Using NLTK, I wrote a script to automatic thematic coding of engagement data (comments and reposts) on STEM Next posts and group them into larger themes to measure thematic trends over time

Sentiment Analysis

Wrote a python script that uses Roberta model best suited to this type of engagement data that automatically assigns each engagement data point between -1 and 1 with 0 as neutral. Allows STEM Next to track the sentiment of engagement data over years, thus tracking how audience opinions and thoughts around STEM Next issues fluxuates over time.

Interactive Map

To measure growth of Youth, Educators, Demographics, and other catagories relevant to funders and programs I developed an interactive map that allows the user to filter by year, state, and measure, and average.
