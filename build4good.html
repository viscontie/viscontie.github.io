Measuring Narrative Change

Project Outcome

Dashboard and protocol that measures narrative change through analyzing and visualizing social media exports and state reported data, sentiment analysis of social media engagement, and thematic trend analysis using NLP tools.

Future Use

Built a protocol for updating the data every 6 months so that with each new batch of social media exports, STEM Next can upload new CSV files to autogenerate new analyses and graphs and track narrative change over years.

Impact

Created a system for tracking narrative change at STEM Next and helped them demonstrate progress toward their goal of reaching 1.5 million kids.

Deliverables

My project included:

  • Website displaying and contextualizing narrative change data for STEM Next employees, partners, and funders.
  • Protocol for updating new data files every 6 months to automatically update Tableau visualizations, run sentiment analysis scripts, and thematic coding scripts.
  • Hour-long presentation on August 20th to STEM Next employees about my findings and the trajectory of measuring narrative change moving forward.

Technical Skills

  • NLP (Roberta) for automated thematic analysis
  • Sentiment Analysis using NLP
  • Python, Pandas, Matplotlib for data analysis & visualization
  • Tableau visualizations and dashboard
  • Website/Dashboard design

Soft Skills

  • Communicating with a nontechnical audience
  • Project management
  • Adaptability: pivoting project due to social media privacy policies
  • Collaboration with stakeholders
  • Framing data to tell the story of narrative change

Thematic Analysis

Using NLTK, I wrote a script to automatically code engagement data (comments and reposts) on STEM Next posts and group them into larger themes to measure thematic trends over time. Check out my code.

Sentiment Analysis

Wrote a Python script using the Roberta model to assign sentiment scores (-1 to 1) to engagement data. This allows STEM Next to track sentiment trends over time and understand how audience opinions evolve. Check out my code.

Interactive Map

To measure growth of Youth, Educators, Demographics, and other categories relevant to funders and programs, I developed an interactive map that allows the user to filter by year, state, and measure, and view averages.
